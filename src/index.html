Countdown

Made in JavaScript

This project:


  1. Is mobile responsive
  2. Stores the countdown in a local storage
  3. Has a video background
  4. Has text animation when the countdown is finished

How it works


On the splash page, enter a title for something you want to count down to. You can either type in the date or use a handy date picker.

The cool thing about this project is that it stores the countdown information in local storage. It allows you to come back to the page another day and the countdown will still be running.

When the countdown is over, it shows „Countdown Complete!" text animation, the title of the countdown, and the date that it finished. There is also a button „New countdown".

Sprint Math Game

Made in JavaScript

The project involves:


  1. Timing events (countdown)
  2. Random correct and incorrect multiplication equations
  3. Random ratio of correct and incorrect answers
  4. Three different views within the game container:
    • the splash page,
    • the game page,
    • the score page,
  5. The best scores stored in local storage

How it works


First, select one type of round, then click „start" and you will see a countdown that’s accomplished using set timeouts.

On the game page, the first equation is highlighted. During the game, it scrolls down automatically to the next equation. It starts a timer when you click the button for the first time, and it stops, when you reach the end.

On the score page you can see a final time, which contains the base time and penalties (0.5 sec for each mistake).

Ping Pong

Made in JavaScript

This project:


  1. Shows how canvas can be animated
  2. Allows you to play against a very basic AI
  3. Has smooth animation
  4. Includes two different views:
    • the game page,
    • the gameover page,

How it works


I recreated the classic pong game in the browser. It starts in a loop, and the computer doesn’t move until you do.

When you enter into the play area, the cursor disappears, and you’re taking over the bottom paddle. As soon as you make contact with the ball, you’re actually able to play against a basic computer AI.

The score is updated and whenever someone scores a point, the ball resets in the center again. If one player reaches 5 points, you will see a gameover screen that tells you who won. It also has a „play again" button.

Drag & Drop

Made in JavaScript

This project:


  1. Uses a drag and drop API
  2. Is stored in local storage
  3. Allows you to edit and delete elements

How it works


The Kanban board is divided into four different columns: backlog, in progress, complete, and on hold. The items or tasks in each column are able to be moved between columns.

If you click on the item, the column lights up, and if you drag it over a different column, it also lights up to show that you can let it go there.

You can click „+Add item" in the column, fill out some text, and press „Save Item". You can also edit every item—just click it, change, and click outside of that box. To delete the item, click it, backspace everything, and click outside of it.

Picture in Picture

Made in JavaScript

This project:


  1. Uses a screen capture API
  2. Creates a virtual second screen
  3. Allows you to move and resize itself

How it works


First, open the tab that you want to use as a picture in a picture. Go to the screen with the Picture in Picture project and refresh it.

The display media method in the screen capture allows you to select what screen, application, or tab you can use. Pick the thing that you want to use and click „share" button.

It will jump back over to the screen that we’re sharing. Return to the project and press „start" to initialize the Picture in Picture. You can move it, resize it, and it will stay on top of everything, allowing you to have a virtual second screen.

Calculator

Made in JavaScript

This project:


  1. Is mobile responsive
  2. Allows you to continue operations
  3. Doesn’t let you enter multiple decimal points
  4. Shows a scroll bar when you enter a large number

How it works


You have the ability to perform basic operations, see the result, and then continue operations on these final values. The clear button can reset it to zero.

When you enter a very large bunch of numbers, a scroll bar will show up, allowing you to see everything.

You are not able to enter more than one decimal point, and once you have pressed operator, you can’t add a decimal point either.
